## FAQ{#faq}

#### Discuz 包含哪些系列？

Discuz 有两个系列的产品：Discuz!X 和 Discuz!Q。

#### [Discuz!Q](https://discuz.com/) 与 Discuz!X 有什么关系和区别？

从品牌上讲，DiscuzQ 是全新架构的 Discuz。但从代码角度看，它们完全不一样。Discuz!Q 的前后端完全分离，后端基于 Laravel，前端基于 Vue.js 和 uni-app，易于二次开发和扩展。
