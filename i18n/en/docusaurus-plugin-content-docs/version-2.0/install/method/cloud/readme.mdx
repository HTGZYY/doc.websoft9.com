## 云资源建议{#howbuy}

Websoft9 的云资源选购经验：

- 地域：理论上“网站用户在哪里，服务器就选哪里”，但实际同一个国家不同地域访问差异不大
- 带宽：带宽选择最大速度，按流量计费是最佳模式（不建议包干带宽模式）
- 内存和 CPU：资源匮乏时，优先保障内存
