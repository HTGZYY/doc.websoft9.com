### 部署 TomEE 应用{#deploy}

以 Tomcat 官方示例包来展示部署过程：

1. 进入 TomEE 容器

2. 进入 web 应用目录并下载示例 war 包
    ```
    cd /usr/local/tomcat/webapps && wget https://tomcat.apache.org/tomcat-10.0-doc/appdev/sample/sample.war
    ```

3. TomEE 会自动解压包，Websoft9 控制台可访问 URL
