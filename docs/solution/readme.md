# 概要

Websoft9 多应用托管平台是不仅仅是一个技术的 PaaS 平台，它从业务的视角看是一个应用的分发与汇聚平台。  

正因为有了多个应用这个属性，使得它非常符合组织创新的原理：**使用工具组合是提升效率的最简单有效的途径**。
