# 安装准备

## 资源

Websoft9 产品天然为云原生而生。


### 云服务器

首先用户需准备自己的云服务器。

* **操作系统**: Red Hat, CentOS, Debian, Ubuntu 等主流 Linux 操作系统
* **公有云**: AWS, Azure, Google Cloud, 阿里云, 腾讯云, 华为云等20多个全球主流云
* **私有云**: KVM, VMware, VirtualBox, OpenStack 等主流虚拟化架构
* **CPU架构**: Linux x86-64, ARM 32/64, x86/i686
* **内存**: 最低1GB
* **CPU**: 最低1核
* **存储**: 最少20GB
* **Swap分区**: 建议2GB
* **带宽**: 100M以上体验更流畅
* **公网IP**: 有独立的公网IP

### 域名

域名不是必须的，但如果您计划给应用配置域名，那么我们建议您尽量在安装之前准备好。

## 操作

1. 在云控制台获取您的 **服务器公网IP地址** 
2. 在云控制台安全组中，检查 **Inbound（入）规则** 下的 **TCP:80** 端口是否开启
3. 若想用域名访问 Graylog，请先到 **域名控制台** 完成一个域名解析
