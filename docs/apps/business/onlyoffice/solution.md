# 场景方案

ONLYOFFICE 可以与其他的软件平台**集成**一起使用，解决 构建企业ERP系统 过程中的各种[场景问题](#)。

## 与网盘集成

ONLYOFFICE Document Server 支持被 ownCloud, Nextcloud, Seafile 等网盘软件集成，用于文档预览与编辑。
