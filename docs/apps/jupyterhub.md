## 入门指南{#guide}

### 初始化{#wizard}

1. Websoft9 控制台安装 JupyterHub 后，通过【我的应用】管理应用，在**访问**标签页中获取登录信息

2. 如果你想增加其他用户，可按照下面命令执行：

  ```
  docker exec -it jupyterhub bash
  adduser youruser
  ```
