# 原理学习

[nopCommerce]([nopcommerce.com](https://www.nopcommerce.com))是.NET领域最优质的企业级商城平台，适合每个商家的需要：它强大的企业和小型企业网站遍布世界各地的公司销售实体和数字商品。nopCommerce是一个透明且结构良好的解决方案，它结合了开源和商业软件的最佳特性。在全球拥有数万名开发者，为你提供源源不断的插件扩展。
