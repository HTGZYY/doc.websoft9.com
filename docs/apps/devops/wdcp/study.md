# 原理学习

[WDCP](https://www.wdlinux.cn/wdcp/) 是国内知名的 Linux 面板和主机管理系统，自带LAMP（LNMP），支持PHP多版本自由切换，功能全面，包括管理文件、查看日志、配置应用服务器、备份数据库、配置域名等，被广泛用于多网站管理与维护。
