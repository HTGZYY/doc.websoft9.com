# 原理学习

[Gogs](gogs.io)是一款极易搭建的自助 Git 服务，相对于GitHub而言Gogs更加轻量级。任何 Go 语言 支持的平台都可以运行 Gogs，包括 Windows、Mac、Linux 以及 ARM。
