# 原理学习

[WampServer](http://www.wampserver.com/?lang=en) 是一个 Windows 环境下的 Apache+PHP+MySQL/MariaDB 组合，由法国人维护的开源项目，拥有简单的图形和菜单安装和配置环境，支持 PHP 多版本切换。支持22种语言，其中包括简体中文和繁体中文。
