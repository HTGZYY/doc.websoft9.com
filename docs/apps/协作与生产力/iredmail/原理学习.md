# 原理学习

[iRedMail](https://www.iredmail.org/) 是一个基于Linux/BSD 系统的开源的、功能完备、成熟的邮件服务器解决方案。只需几分钟，iRedMail 即可为您部署一台完全基于开源软件、功能完善的邮件服务器。
