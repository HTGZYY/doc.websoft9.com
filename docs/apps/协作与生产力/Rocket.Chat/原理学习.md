# 原理学习

Rocket.Chat 是一个类似 Slack 的交流平台，可以将外部客户，团队和其他生态相关者聚集到一个平台中的交流平台。自2015年面世以来，迅速成长为出色的开源项目之一。目前，Rocket.Chat广泛应用在银行，非政府组织，初创企业和政府组织，通过自定义其外观和风格，用户可以在云上或通过在本地托管自己的服务器来设置 Rocket.Chat，用户可以安全地管理数据。
