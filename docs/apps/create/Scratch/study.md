# 原理学习

[Scratch](https://scratch.mit.edu/) 是一款由麻省理工学院(MIT) 设计开发的一款面向少年的简易编程工具，scratch已经是少儿编程行业的基础软件。使用 Scratch，你可以编写属于你的互动媒体，像是故事、游戏、动画，然后你可以将你的创意分享给全世界。Scratch3.0是一个部署在服务器上的Web版本，有浏览器就可以使用Scratch。
