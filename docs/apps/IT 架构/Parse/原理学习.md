# 原理学习

[Parse Server ](https://parseplatform.org/) 是一个用Node.js编写的 Serverless 后台程序。它抽象了后端开发常用功能，并以服务端的方式对外提供API供第三方应用程序使用。针对不同环境、不同语言，官方同时提供了客户端SDK以简化您的开发工作。

Parse Server 提供的基础功能包括:

- 用户的登录注册
- 用户身份的认证
- 数据存储 && 灵活查询
- 文件存储
- 实时查询
- 消息推送
- 缓存服务
- 与云平台很好的对接
- 自定义业务逻辑与Hook机制
