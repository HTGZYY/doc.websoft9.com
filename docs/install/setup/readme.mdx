# 安装后配置

安装 Websoft9 的产品后，用户朋友最想做的事情肯定是立即能够看到软件的界面，投入使用。   

但现实却是一个应用真正的能让用户快速使用上，它是需要一些必不可少的配置的：

## 准备{#prepare}

* [操作云服务器](../setup/vm)：连接服务器和操作云控制台、获取服务器公网IP、开启安全组端口等
* [获取账号密码](../setup/credentials)：
* [初始化向导](../apps)：本地浏览器访问 *http://服务器公网IP* 可进入初始化向导

## 配置域名{#domain}

配置域名最好早于**初始化向导**，这样可以避免修改应用的 URL。下面是**“域名五步”** 操作：

1. [域名注册](../dns#domain)
2. [域名实名制认证](../dns#domain)
3. [域名备案](../dns#domain)
4. [域名解析](../dns#domain)
5. [域名绑定](../dns#domain)

* 更换数据库
